#Project Title: Smart Toolkit
# A sleek, premium toolkit application built with React Native and Expo

The Tech Stack: 
# Framework: React Native (Expo SDK)
# Language: TypeScript
# Navigation: Expo Router (File-based routing)
# Storage: AsyncStorage for persistent data
# Icons: Expo Vector Icons (Ionicons)

# Stage 1 New Challenge: Stage 1 required scaling my app's architecture to handle multiple features (Unit Conversion + Task Management) while maintaining a consistent UI/UX. This task improved my understanding of the React Lifecycle and how to manage local device storage efficiently. Implemented offline data persistence using @react-native-async-storage/async-storage to ensure user data remains available across app sessions without an internet connection.

# Key Features
✔ Stateful Task Manager: Full CRUD functionality with local persistence using @react-native-async-storage/async-storage .
✔ Advanced Converters: Precision-based conversion tools for Distance, Temperature, and Currency.
✔ Branded UI System: Custom BrandedHeader component for consistent navigation and typography.
✔ Platform-Specific Styling: Optimized shadows and elevations using the Platform API for iOS and Android.
